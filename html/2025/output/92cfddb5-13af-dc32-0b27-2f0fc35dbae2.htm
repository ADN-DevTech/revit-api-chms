Property Value

RowHeightOverrideOptions
  • None to disable row height override.
  • All enables row height override for all the body rows in the schedule.
  • ImageRows enables row height override for all the schedule body rows that contains images or custom graphics.
Exceptions
ExceptionCondition
ArgumentOutOfRangeException When setting this property: A value passed for an enumeration argument is not a member of that enumeration
Remarks

Setting this property to anything but None will allow setting the RowHeight property. This is taken into account when the schedule is viewed as a ScheduleSheetInstance on a ViewSheet.
