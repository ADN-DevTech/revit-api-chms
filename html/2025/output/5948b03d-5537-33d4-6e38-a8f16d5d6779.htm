Parameters

saveModified  Boolean
Indicates if the current document should be saved before close operation.

Return Value

Boolean
False if closing procedure fails or if saving of a modified document was requested (saveModified = True) but failed. Also returns False if closing is cancelled by an external application during 'DocumentClosing' event. When function succeeds, True is returned.
Exceptions
ExceptionCondition
InvalidOperationException Thrown when attempting to close the currently active document. Thrown if there are any transactions, sub-transactions or transaction groups which were opened by the API code, and not closed. All of these items must be handled before attempting to close the document. Thrown if saveModified is 'true' and the PathName is not set yet. Thrown if saveModified is 'true' and the saving target file is read only.
InvalidOperationExceptionThrown if this a linked file.
Remarks
The currently active document may not be closed by this function. It can only be closed via Revit's UI.
