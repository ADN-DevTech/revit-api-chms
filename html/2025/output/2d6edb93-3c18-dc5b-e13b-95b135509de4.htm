Assignable0 The DirectShapeType can be used to replace the type of an existing DirectShape.
Nonassignable1 The DirectShapeType cannot be used to replace the type of an existing DirectShape.
Remarks
Only DirectShapeTypes which permit assignability will be shown in the type selector dialog. A DirectShape which uses a DirectShapeType which is not assignable can be copied.
